The TTB

  • All alcoholic beverage labels require government approval
  • The Alcohol and Tobacco Tax and Trade Bureau (TTB) maintains a public record of all applications

What's available

  • Meta information related to the application
  • The application itself
  • Meta information regarding the label(s)
  • The label(s)

Caveats

  • Many exceptions to rules (ex ID number should be sequential for each day)
  • Some applications have missing data (ex missing image)
  • Some applications are poorly taken photos

Finding dominant colors

Understanding images

  • Images are represented as matrices (width, height, channels)
  • Typically there are 3 channels (ex RGB) but there can also be 4 (ex RGBA)
  • But, there are many color spaces and thus many ways to represent color data
    • RGB, Lab, CMYK, HSV, etc.

Treat each pixel as a classification problem

  • KMeans to find clusters
  • Silhouette to find optimal number of clusters
